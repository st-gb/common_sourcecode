C++ compiler suite for Microsoft's "Windows" operating system
=============================================================

There are several compiler suites available for Microsoft Windows.
Here are Open Source ones:
-TDM GCC( http://tdm-gcc.tdragon.net )
-MSYS2's MinGW64:see file "compiler/C++/MSYS2_MinGW64.txt" of Stefan Gebauer's
 (TU Berlin matriculation number 361095) ~"common sourcecode" repository (may be
 located online in http://github.com/st-gb/common_sourcecode )
-Clang( https://clang.llvm.org )
